📊 E-commerce Sales Dashboard (Power BI)

 📌 Project Overview

This project analyzes e-commerce sales data to identify key trends, customer behavior, and business opportunities using Power BI.


🎯 Objective

To build an interactive dashboard that helps stakeholders understand:

* Sales performance
* Profit trends
* Customer segments
* Regional distribution


🛠️ Tools Used

* Power BI
* DAX
* Data Modeling

 🔍 Key Insights

* Sales are slightly declining YoY
* Profit is improving due to better margins
* Consumer segment generates highest revenue
* Office Supplies is the top-performing category
* Standard shipping is preferred by customers


💡 Business Recommendations

* Focus on underperforming regions (Central & South)
* Improve Technology category performance
* Optimize low-performing products
* Introduce faster shipping strategies
